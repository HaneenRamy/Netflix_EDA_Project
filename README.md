This project performs Exploratory Data Analysis (EDA) on the Netflix dataset using Python.  
The goal is to explore patterns, trends, and insights related to content type, genres, duration, and country distribution.

---

## Project Objectives

- Understand the distribution of Movies vs TV Shows
- Analyze content production by country
- Compare movie durations across top countries
- Identify top 5 most common genres
- Explore trends and patterns in Netflix content

---

## Tools & Technologies

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset

The dataset contains information about:
- Title
- Type (Movie/TV Show)
- Country
- Duration
- Genre
- Release Year
- Date Added

---

## Key Analysis Performed

- Data cleaning and preprocessing
- Correlation analysis
- Heatmap visualization
- Country-wise comparison
- Genre frequency visualization

---

## How to Run

1. Clone this repository
2. Open the notebook in Jupyter
3. Install required libraries
4. Run the notebook cells step by step
